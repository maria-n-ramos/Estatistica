# Estatística

Repositório com trabalhos desenvolvidos ao longo da graduação em Estatística.
Reúne exercícios e trabalhos avaliativos de disciplinas cursadas, com resoluções em
Python ou R.

## Áreas e pastas do repositório

### Estatística Descritiva
Atividades relacionadas à organização e resumo de dados, incluindo medidas
descritivas, tabelas e gráficos.

### Probabilidade
Exercícios envolvendo conceitos fundamentais de probabilidade, variáveis
aleatórias e distribuições teóricas.

### Inferência Estatística
Trabalhos voltados à inferência sobre populações a partir de amostras, como
testes de hipóteses, intervalos de confiança e comparação de médias.

### Regressão e Correlação
Estudos sobre relação entre variáveis, regressão linear simples e múltipla,
correlação e interpretação dos resultados.

### Estatística Multivariada
Trabalhos envolvendo métodos multivariados, incluindo:
- Análise de Componentes Principais (PCA)
- Análise Fatorial
- Análise de Correlação Canônica
- Análise Discriminante
- Análise de Agrupamentos
- Análise de Correspondência

### Métodos Estatísticos Aplicados
Aplicações estatísticas integrando técnicas de diferentes áreas, utilizando
conjuntos de dados reais ou simulados.

## Observação
Os materiais correspondem a atividades acadêmicas desenvolvidas durante a
graduação em Estatística.
